### XDNR

----------

#### compile and run

```
gcc xdnr.c -o xdnr
chmod +x xdnr
./xdnr
```

----------

XDNR is a X0R Cryptor along with DEC/N0T/R0R encoder plus random byte insertion encoder, 
which generates null free encrypted and encoded shellcodes
